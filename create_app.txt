How to create a new react app

step 1: create a index.html file
step 2: define a root element where react element will be created
    >>  <div id="root">
            <h3>root element</h3>
        </div>
step 3: link app.js to index.html file
    >>  <script type="module" src="app.js"></script> (place just above </body>)

step 4: setup an app.js file where we play with React

step 5: import react from node modules library in app.js

        >> import React from "react";
        >> import ReactDOM from "react-dom/client";

step 6: define a root element in the ReactDOM

        >> const root = ReactDOM.createRoot(document.getElementById("root"));

step 7: Now we can define the react elements and functional/class based components in react
        >> const Applayout =>{
            <>
                <h2> Hello World </h2>
            </>
        }

step 8: To render that component inside root element , run following command        
        >> root.render(<Applayout />);


Step by step process to setup react app

1. Install NodeJs Locally
2. npm init or npm init -y (saves from entering all details about project)
3. npm install react or npm i react
4. npm i react-dom
5. npm i -D parcel (to install parcel as dev dependeny) or npm i --save-dev parcel
6. npx parcel index.html (run the app using the command)
	npm used when installing a package, npx used for executing commands and packages
7. making shorthand for commands in package.json  under script section
	start : "parcel entrypoint.exe ", (to start the app)
    this will allow use to use command like npm run start for starting the app
9. npm start / npm run start/ npx parcel index.html are same to run project
